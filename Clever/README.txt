Clever database API.
You will need mysql and Python to run this program

We built this project to highlight some of the improvements that Clever can make to improve their current application

This is a project that interracts with an hypothetical database and extracts and edits it, by the given input.
for the company Clever.

It is an API and therefore will not work without the database.

To run the program be sure to enter the correct values in the mydb variable, by opening it in your preferred text editor
These values should be found in your mysql program.
Afterwards, you should be able to run the program in your terminal.


When running the program; it can currently only accept entries as numbers, and nothing more should be needed.
It will function with any amount of data in the given tables so feel free to add/edit data as you like.
